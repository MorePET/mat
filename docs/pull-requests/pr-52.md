# [PR 52](https://github.com/MorePET/mat/pull/52) feat!: 3.0 vis cutover — `properties.pbr` removed, `.vis` canonical, mat-vis-client 0.4.x

## Description

py-materials 3.0 — the vis cutover. Consolidates all PBR state under `material.vis`, removes `material.properties.pbr` / `PBRProperties` / the `pbr=` kwarg / the `[<material>.pbr]` TOML section with no deprecation cycle, and raises the `mat-vis-client` floor to `>=0.4.0`.

Full migration story in [`docs/migration/v2-to-v3.md`](docs/migration/v2-to-v3.md). Rationale for skipping 2.3 deprecation-only in [#40](https://github.com/MorePET/mat/issues/40).
