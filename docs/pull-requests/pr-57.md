## Description

Three coordinated tightenings for the release pipeline post-3.0:

1. **Disable rs-materials in release-please** until #43 enables crates.io publish. Stops the recurring false-bump Release PRs (#50, #55).

3. **Document the scope taxonomy** — \`(py)\` / \`(rs)\` / \`(data)\` / \`(vis)\` / \`(deps)\` / \`(ci)\` / \`(docs)\`.
