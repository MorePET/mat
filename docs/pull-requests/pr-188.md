## Why 3.10.0 not 3.9.1

#186 originally targeted 3.9.1 (patch — pure metadata fix). But by the time it merged, the four feats above had already landed on main without a release cut, so the version delta from `v3.9.0` includes 4× `feat:` work. Per semver, that's a minor bump regardless of #186's scope. Shipping as 3.9.1 would have either omitted the four feats from the changelog (lying release notes) or relabeled feat-level work as a patch (lying semver).
