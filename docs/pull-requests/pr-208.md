# [PR 208](https://github.com/MorePET/mat/pull/208) feat(data): add 4 technical ceramics (AlN, LTCC 951, sapphire, Si3N4) (#134, #135, #136, #137)

## Summary

Adds four primary-source-cited technical ceramic entries to `src/pymat/data/ceramics.toml`, with per-property `_sources` provenance:

- **Aluminum nitride (AlN)** — sintered AlN-170 commercial grade. Vendor scalars from CoorsTek; Slack 1987 (DOI `10.1016/0022-3697(87)90153-3`) cited as the intrinsic-thermal-conductivity review (~285 W/(m·K) single-crystal upper bound vs the 170 W/(m·K) commercial engineering scalar). Anisotropic CTE [a,a,c] = [4.2e-6, 4.2e-6, 5.3e-6] / K. Closes #134.
- **LTCC DuPont 951** — DuPont GreenTape 951 public TDS (MCM951, 7/2011). All scalars verbatim from the "Typical Tape Properties" table. License = `proprietary-reference-only`. Closes #135.
- **Sapphire (single-crystal Al2O3)** — distinct from polycrystalline `alumina`. Refractive index n_o = 1.768 at sodium D-line from Malitson & Dodge 1972 (DOI `10.1364/JOSA.62.001405`). Mechanical/thermal scalars from Crystran technical brief; hardness/toughness from Dobrovinskaya 2009 textbook (ISBN 978-0-387-85695-7). Anisotropic CTE [a,a,c] = [5.0e-6, 5.0e-6, 5.6e-6] / K. Closes #136.
- **Silicon nitride (Si3N4)** — sintered SSN/DPSSN grade, the most common engineering Si3N4. All scalars verbatim from the Superior Technical Ceramics 2021 datasheet (ASTM-method-stamped). Riley 2000 (DOI `10.1111/j.1151-2916.2000.tb01182.x`) cited as the materials-class review. Closes #137.

### Primary citations

- **Riley 2000** — `10.1111/j.1151-2916.2000.tb01182.x`, J. Am. Ceram. Soc. 83(2), 245 — Si3N4 materials-class review.
- **Malitson & Dodge 1972** — `10.1364/JOSA.62.001405`, J. Opt. Soc. Am. 62, 1405 — sapphire refractive index (sodium D-line).
- **Dobrovinskaya, Lytvynov, Pishchik 2009** — Springer ISBN 978-0-387-85695-7 — sapphire hardness/toughness/max-service-temp.
- **DuPont GreenTape 951 TDS** (MCM951, 7/2011) — vendor datasheet; all LTCC scalars.
- **Superior Technical Ceramics 2021** — DPSSN/SSN datasheet; all Si3N4 scalars (ASTM-method-stamped).
- **CoorsTek AlN-170 brochure** — vendor datasheet; AlN engineering scalars.

- **Wikipedia AlN article** (CC-BY-SA-4.0) — AlN density, melting point, Cp conversion, anisotropic CTE.

### Judgment calls

- **AlN thermal conductivity**: quoted commercial value 170 W/(m·K) (CoorsTek AlN-170), not the ~285 W/(m·K) single-crystal theoretical max. Slack 1987 attached as a `thermal._review` source pointing at the upper bound.
- **Sapphire anisotropy**: schema has one scalar per property. Picked c-axis (extraordinary) where it differs from a-axis (thermal expansion 5.6e-6 / K parallel-c canonical, perp-c 5.0e-6 / K in note). For dielectric constant we picked the perpendicular-c value (9.4) since that's the most-cited literature scalar; parallel-c (11.5) is in the source note. Anisotropic CTE stored separately in `cte_anisotropic = [a,a,c]`.
- **Sapphire refractive index**: stored ordinary ray n_o = 1.768 at 589.3 nm. Extraordinary ray n_e = 1.760 documented in the source note but not as a scalar — the schema has one `refractive_index` field.
- **Si3N4 grade**: picked sintered Si3N4 (SSN/DPSSN), the most common engineering grade. Hot-pressed and reaction-bonded grades have different properties; not added.
- **LTCC shrinkage**: schema has no shrinkage field, so 12.7% xy / 15% z plus the 10 GHz dielectric constant and surface roughness live in `[ltcc951.custom]` to survive the loader without a schema change.
- **Sapphire density**: Crystran reports 3.97 g/cc; Wikipedia range 3.98–4.06 g/cc. Picked 3.98 (rounded to match the long-standing handbook value).

### Intentionally omitted (no schema field, or no primary scalar)

- **AlN single-crystal thermal conductivity** (~285 W/(m·K), Slack 1987) — engineering canonical is the commercial value; theoretical max retained as `thermal._review`.
- **Sapphire bulk dispersion table** — that's #201's refractiveindex.info enricher territory; sapphire isn't yet in scope.
- **Sapphire dielectric constant parallel-c** (11.5) — schema has one `dielectric_constant` scalar; perpendicular-c (9.4) is canonical.
- **Si3N4 specific heat** — STC datasheet doesn't tabulate Cp; cited Riley 2000 review for the ~700 J/(kg·K) scalar.
- **LTCC 951 youngs_modulus / flexural_strength provenance**: confirmed verbatim transcribed from datasheet, no separate primary paper.
