# [Issue 170]: [LBNL Scintillator Library — discovery index (cite primary)](https://github.com/MorePET/mat/issues/170)

[scintillator.lbl.gov](https://scintillator.lbl.gov/) is the closest thing to a "scintillator zoo" — Derenzo/Boswell/Goldblum/Laplace, DOE-funded. Each entry cites original measurement papers. License not explicitly stated; treat as **discovery index, not a redistribution source**.

Action: use to find primary papers for each scintillator we add (GAGG, LaBr3, CeBr3, SrI2, LSO). Cite the primary paper in `_sources`, not the LBNL site.

2024 reference paper: Laplace et al., NIM A — escholarship.org/uc/item/4rx6b1k3 (open-access preprint).
