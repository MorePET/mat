# [Issue 151]: [Schema: mechanical sub-table additions](https://github.com/MorePET/mat/issues/151)

Add to `mechanical`: `flexural_modulus` / `flexural_strength` (MPa, plastics & ceramics datasheets), `fatigue_limit` (MPa, cyclic gantry loads), `cte_anisotropic = [a, b, c]` (ppm/K, for non-cubic crystals like sapphire/quartz), `creep_rate` table (T, stress → strain/hr, cryostat long-term loads), `cti` (V, PCB substrate selection).
